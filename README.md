[**Try out the demo!**](https://benjaminiserman.github.io/diabetes-tools/)

## Build and run

### To Run Locally:

Clone the repository and perform the following command line actions:

```bash
$> cd diabetes-tools
$> npm install
$> npm run start
```

Open [http://localhost:3000](http://localhost:3000) in browser.
